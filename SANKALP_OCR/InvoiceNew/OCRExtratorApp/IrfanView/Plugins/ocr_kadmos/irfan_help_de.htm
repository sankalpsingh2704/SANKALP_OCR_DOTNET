KADMOS Hilfedatei für Irfanview, Deutsch
User language setting has to be done in Irfanview under Options, Properties/Settings, Languages.
Die Bediener-Sprache ist in Irfanview unter Options, Properties/Settings, Languages einzustellen.

Wurde mit Irfanview ein Bild geladen und die Kadmos-Zeichenerkennung gestartet, so öffnet sich ein neues Fenster mit dem zu erkennenden Bild auf der linken Seite.

Die Zeichenerkennung wird gestartet, wenn man mit der Maus einen der gelb unterlegten Textbereiche anklickt oder aber mit der Maus ein Gummiband um einen Bereich seiner Wahl zieht. Ein Editorfenster mit dem Ergebnis öffnet sich. Darin kann ediert und das Ergebnis gespeichert werden.


Konfiguration:
Hier können Parameter für die Zeichenerkennung gesetzt werden, wie etwa die Zeichenhöhe, Zeilenteilung, oder Art der Schmutzentfernung. Zudem kann der Zeichensatz des für die Erkennung geladenen Klassifikators eingeschränkt werden.


Textsprache wählen:
Bevor eine Erkennung gestartet werden kann, muss zuerst die Textsprache eingestellt werden. Bei Klick auf den Pfeil des Textsprachen-Auswahlfensters erscheinen alle installieren Sprachen. Ist die gewünschte Sprache nicht enthalten, so kann diese mit "Download weitere Sprache" installiert werden.


Download weitere Sprache:
Es öffnet sich ein Fenster "KADMOS Klassifikator Herunterladen". Nach einem Klick auf den Pfeil des Sprachauswahlfensters erscheinen alle wählbaren Länder.
